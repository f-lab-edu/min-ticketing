# Min-Ticketing

---


#### 01. 소개

> Min-Ticketing은 온라인 공연 좌석 예매 티케팅 서버 프로젝트입니다.


#### 02. 기능

- 자체, Google 로그인 및 회원가입(회원가입 시 이메일 인증)
- 공연 정보 및 좌석 정보 조회
- 공연 좌석 예매 
- 결제(Toss Payments API 사용)


#### 03. 사용 기술

- `kotlin`
- `Spring Boot`, `Spring Data JPA`
- `MySQL`, `Redis`
- `koTest`, `mockk`
